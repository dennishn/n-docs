Table Sizing

We have seen alot of attempts to utilize the grid system offered by Bootstrap to control the sizing of table columns. Unfortunatly defining widths on table columns is near impossible. What you can do however is to define a width attribute on the ths or tds with a value in pixels.
What you need to understand is that even though defining a width in pixels, a table will not size the cell to this specific width, but you can use it to prioritize a column width to be higher or lower than others like in the following example.

Responsive Tables

Create a responsive table by wrapping a .table in a <div> with the class .table-responsive. If the .table-responsive is wider than it's parent element it will add a horizontal scrollbar.
