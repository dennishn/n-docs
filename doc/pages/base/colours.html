Colors

Dennis skal docce mixins

In bootstrap only one "brand" color is available: $brand-primary. In nStack we need alot more, and as such we have chosen to disband bootstraps way of using colors in favour of our own more complex solution... The color system consists of 2 colored palettes, and a selection of grays.

Text colors

Text colors are designed to be used complementary to the opposite main color backgrounds.

Text shades

Shades are designed to be used with contextual backgrounds, but can work equally well on the primary and secondary color backgrounds. Text can be shaded light and dark, each with 3 variations: primary, secondary, hint (hint is for hint-text, disabled etc.).

Colour palettes

A palette consists of a "base" color, and 4 variatioins in lighter and darker shades.

  • Primary Color
  • Secondary Color
  • Info Contextual Color
  • Success Contextual Color
  • Warning Contextual Color
  • Danger Contextual Color
  • Grays
